# 게임오버 씬 및 레벨 선택 복귀 구현

## 작업 내용

- `Assets/Scenes/GameOver.unity` 게임오버 씬을 새로 만들었다.
- 기존 프로젝트의 `Sprite-00012` 9분할 버튼 이미지, 도트 폰트, 토끼·거북이 스프라이트를 재사용해 다른 화면과 어울리는 도트 UI로 구성했다.
- `레벨 선택으로 돌아가기` 버튼을 누르면 `Level` 씬으로 이동하도록 구현했다.
- 오프라인 실행에서는 일반 씬 이동을 사용하고, 네트워크 연결 중에는 클라이언트의 요청을 호스트가 받아 모든 플레이어를 `Level` 씬으로 이동시킨다.
- `GameOver` 씬을 Build Settings에서 `Level` 씬 다음에 등록했다.
- 작업 전 열려 있던 `InGame_portaldev` 씬과 기존 에셋은 유지했다.

## 확인

- Unity 플레이 모드에서 게임오버 화면이 정상적으로 렌더링되는 것을 확인했다.
- Unity Console 컴파일 오류 0개를 확인했다.
- `Assembly-CSharp.csproj` 빌드 결과 오류 0개이며, 기존 Unity 패키지 참조 버전 경고 2개만 남아 있다.
- 씬의 버튼, 상태 문구, 네트워크 컨트롤러 참조와 `Level` 대상 설정을 확인했다.

## 팀 테스트 항목

- 호스트와 클라이언트가 함께 게임오버 씬에 들어간 뒤 어느 쪽에서 버튼을 눌러도 두 화면이 `Level` 씬으로 이동하는지 확인한다.
