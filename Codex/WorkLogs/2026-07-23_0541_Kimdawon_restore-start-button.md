# Start 씬 도트 버튼 복구

## 원인

- `Start` 씬의 `Button_Sample` 오브젝트는 삭제된 상태가 아니라 비활성화된 상태였다.
- 시작 메뉴 코드는 비활성 오브젝트도 찾을 수 있었지만, 오브젝트 자체가 꺼져 있어 화면에 버튼이 표시되지 않았다.

## 변경 내용

- `Assets/Scenes/Start.unity`에서 `Button_Sample`을 다시 활성화했다.
- 시작 메뉴가 버튼을 구성할 때 해당 오브젝트를 자동으로 활성화하도록 안전장치를 추가했다.
- 타이틀 오브젝트와 `Sprite-00011` 에셋은 수정하지 않았다.
- 버튼은 기존 `Sprite-00012` 9분할 이미지와 Neo둥근모 참조를 그대로 사용한다.

## 확인 결과

- `Start.unity`가 버튼이 정상적으로 활성화된 `bak_Start.unity`와 동일한 상태임을 확인했다.
- `Button_Sample`의 스프라이트와 TMP 글꼴 참조가 유지됨을 확인했다.
- C# 컴파일 검증을 통과했다.
- 플레이 모드 확인은 팀에서 직접 진행하기로 한 요청에 따라 실행하지 않았다.
