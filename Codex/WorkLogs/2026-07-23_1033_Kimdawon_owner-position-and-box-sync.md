# Kimdawon 작업 내용 - 소유자 위치 및 상자 동기화

## 문제

- 토끼 물리는 소유자 컴퓨터에서 정상 동작하지만 애니메이션만 상대에게 전달되고 실제 위치가 전달되지 않음.
- 클라이언트가 거북이를 소유한 경우 서버가 오래된 거북이 위치로 상자 위치를 계산할 가능성이 있음.

## 변경

- 플레이어 소유자가 Rigidbody2D의 위치와 회전을 Owner 쓰기 권한 NetworkVariable로 직접 전송함.
- 소유하지 않은 복제 플레이어는 전달받은 위치와 회전을 Kinematic Rigidbody2D에 적용함.
- 서버의 상자 검색 및 들고 있는 상자 위치 계산이 동기화된 거북이 위치와 방향을 사용하도록 변경함.
- 기존 애니메이션 NetworkVariable 동기화는 유지함.

## 기대 결과

- 토끼가 호스트 또는 클라이언트여도 토끼 소유자의 실제 물리 위치가 상대 화면에 표시됨.
- 클라이언트 거북이가 상자를 들고 이동해도 서버와 다른 클라이언트 화면의 상자가 같은 위치를 따라감.

## 확인

- Unity 강제 재컴파일 후 Console 오류 0개.
- 호스트·클라이언트 동시 실행 동작은 사용자 테스트 예정.
