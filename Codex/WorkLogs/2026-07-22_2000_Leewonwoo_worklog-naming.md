# 작업 기록 작성자 이름 변경

## 작업 목적

`leewonwoo` 브랜치에서 Codex가 생성하는 작업 기록 파일명에 `Leewonwoo`가 포함되도록 협업 지침을 변경한다.

## 변경한 파일

- `Codex/AGENTS.md`
- `Codex/WorkLogs/2026-07-22_2000_Leewonwoo_worklog-naming.md`

## 핵심 변경 내용

- 작업 기록 파일명에 반드시 포함할 이름을 `Kimdawon`에서 `Leewonwoo`로 변경했다.
- 권장 파일명 형식을 `YYYY-MM-DD_HHMM_Leewonwoo_<작업명>.md`로 변경했다.

## 확인 또는 테스트 결과

- 지침의 작성자 이름과 권장 파일명 형식에 `Leewonwoo`가 반영된 것을 확인했다.
- 문서만 변경했으므로 Unity 실행 테스트는 필요하지 않다.

## 남은 작업이나 주의 사항

- 기존 `Kimdawon` 작업 기록은 과거 이력 보존을 위해 이름을 변경하지 않았다.
